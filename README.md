# Plumber Business 

A complete digital transformation for a Brisbane plumbing business.
Delivered as a freelance engagement covering full-stack development,
SEO strategy, and business process automation.

---

## 🏆 Outcome

| Metric | Result |
|--------|--------|
| Website traffic increase | 825% |
| Lead conversion rate | 55% |
| Client acquisition | Multiple referral clients within weeks of launch |

---

## 🔍 Scope of Work

**1. Digital Audit**
- Full SEO gap analysis
- Competitor benchmarking
- Traffic source mapping
- Delivered MoSCoW prioritised roadmap with defined acceptance criteria

**2. Website Redesign & Development**
- Full-stack rebuild with modern responsive UI
- Emergency booking workflow for after-hours calls
- Third-party API integrations (booking, maps, contact)
- SSL implementation and Google Business Profile setup

**3. Technical SEO**
- On-page SEO optimisation
- Page speed improvements
- Mobile responsiveness
- Structured data implementation

---

## 🛠 Tech Stack

**Framework:** Next.js  
**Styling:** Tailwind CSS, responsive UI/UX  
**Integrations:** Third-party booking APIs, Google Business Profile  
**Security:** SSL certificate setup  
**SEO:** On-page optimisation, structured data, sitemap  
**Analytics:** Google Analytics traffic monitoring
